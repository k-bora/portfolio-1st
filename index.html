프로젝트 설명
팀 프로젝트로 “따순기미”라는 빵집(파주 금촌점)의 홈페이지를 제작하게 되었습니다. 저는 클라이언트와의 미팅, 정보취합 (제안서 작성, 스토리보드 작성)과 PC 화면의 html5, css3, javascript, jQuery 코드를 담당하여 진행하였습니다. 웹서버는 Apache 2.2를 사용하고 데이터베이스 MySQL 5.7와 PHP 버전7.2의 닷홈 호스팅웹을 사용하여 제작하였습니다.


제안서 설명
시장조사, 경쟁사분석부터 시작해서 시안 3가지를 제안했습니다. 첫 미팅 후 one page로 결정되고, 홈페이지에 들어갈 로고도 원하셔서 로고와 2차 시안까지 넣은 제안서입니다.
